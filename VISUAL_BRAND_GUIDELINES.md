# Premium TechNoir Visual Brand Guidelines

This document establishes the visual identity standards for Premium TechNoir. All design work should reference these guidelines to maintain consistency across all digital and physical touchpoints.

---

## Logo System

### Logo Variants

#### Primary Logo (Full)
- Shield badge icon with integrated checkmark
- "Premium TechNoir" wordmark below or beside
- Minimum width: 120px
- Color: Aqua (#2FE7F2) with Navy gradient

#### Icon Only
- Just the shield badge
- Used when space is limited
- Minimum size: 48px × 48px
- Clear space: Minimum 8px around all sides

#### Horizontal Lockup
- Icon on left, wordmark on right
- Used for headers, websites, applications
- Minimum width: 200px

#### Vertical Lockup
- Icon above wordmark
- Used for social media, small spaces
- Minimum width: 120px

#### Monochrome Variants
- For applications where color isn't possible
- White on dark backgrounds
- Black on light backgrounds

### Clear Space
Minimum 8px of clear space around the entire logo.
Never place logos closer than 8px to other design elements.

### Sizing Guidelines
- **Minimum Size:** 48px (icon only), 120px (full logo)
- **Maximum Size:** Constrained by container or design
- **Aspect Ratio:** Maintain original proportions (never stretch)

### Colors

#### Full Color Logo
- **Icon:** Aqua (#2FE7F2) transitioning to Navy Blue (#001A4D)
- **Text:** White (#FFFFFF) on dark backgrounds
- **Text:** Navy (#08131F) on light backgrounds

#### Reversed (White)
- **Icon:** White (#FFFFFF)
- **Text:** White (#FFFFFF)
- Use on Navy backgrounds

#### Reversed (Black)
- **Icon:** Black (#000000)
- **Text:** Black (#000000)
- Use on light backgrounds

### Incorrect Usage (Never Do This)
- ❌ Rotate the logo
- ❌ Stretch or squash the logo
- ❌ Change colors to other brand colors
- ❌ Add drop shadows or effects
- ❌ Place on cluttered backgrounds without contrast
- ❌ Use outdated wordmarks

## Typography Standards

### Typeface Families

**Headings**
- Primary: Space Grotesk
- Secondary: Sora
- Purpose: Large, bold headlines and navigation
- Characteristic: Modern, geometric, tech-forward

**Body Text**
- Primary: Inter
- Purpose: All body copy, labels, UI text
- Characteristic: Clean, highly readable, universal

### Font Weight Usage

| Weight | Name | Usage |
| --- | --- | --- |
| 400 | Regular | Body text, regular UI |
| 500 | Medium | Labels, UI text, slight emphasis |
| 600 | Semibold | Subheadings, important text |
| 700 | Bold | Headings, strong emphasis |

### Type Scale

```
Display 1: 72px / 86px line-height / Bold (700)
Display 2: 60px / 72px line-height / Bold (700)

Heading 1: 48px / 62px line-height / Bold (700)
Heading 2: 40px / 52px line-height / Bold (700)
Heading 3: 32px / 45px line-height / Semibold (600)
Heading 4: 24px / 34px line-height / Semibold (600)
Heading 5: 20px / 30px line-height / Semibold (600)
Heading 6: 16px / 24px line-height / Semibold (600)

Body Large: 18px / 27px line-height / Regular (400)
Body Medium: 16px / 24px line-height / Regular (400)
Body Small: 14px / 21px line-height / Regular (400)
Body XS: 12px / 18px line-height / Regular (400)

Label Medium: 14px / 21px line-height / Medium (500)
Label Small: 12px / 18px line-height / Medium (500)
Label XS: 11px / 15px line-height / Semibold (600)

Caption: 12px / 16px line-height / Regular (400)
```

### Text Color Combinations

**High Contrast (Maximum Readability)**
- Text: White (#FFFFFF)
- Background: Matte Black (#0D1117) or Midnight Navy (#08131F)
- Use for: Primary body text, headings

**Medium Contrast (Good Readability)**
- Text: Light Gray (#E0E0E0)
- Background: Midnight Navy (#08131F)
- Use for: Secondary text, supporting information

**Low Contrast (Subtle Information)**
- Text: Silver (#C5CBD3) at 70-80% opacity
- Background: Midnight Navy (#08131F)
- Use for: Disabled states, placeholder text

## Layout & Spacing

### Spacing Grid (8px)

All spacing should be multiples of 8px for visual consistency:

- **4px:** Micro-interactions, tight spacing
- **8px:** Padding within components
- **12px:** Small component spacing
- **16px:** Default padding/margins
- **24px:** Section spacing
- **32px:** Major spacing between sections
- **48px:** Large spacing, section breaks
- **64px+:** Page-level spacing

### Container Widths

```
Mobile: 375px (320px-480px)
Tablet: 768px (481px-1024px)
Desktop: 1024px (1025px-1440px)
Wide: 1440px (1441px+)
```

### Padding by Container

```
Mobile:   16px horizontal padding
Tablet:   24px horizontal padding
Desktop:  32px horizontal padding
Wide:     48px horizontal padding
```

### Component Spacing

```
Button Padding: 12px (top/bottom) × 24px (left/right)
Input Padding: 12px (top/bottom) × 16px (left/right)
Card Padding: 16px (mobile) → 24px (tablet/desktop)
Section Margin: 40px (mobile) → 64px (desktop)
```

## Shadow & Depth System

### Shadow Scale

| Name | Value | Usage |
| --- | --- | --- |
| None | none | Flat elements |
| XS | 0px 1px 2px rgba(0,0,0,0.05) | Subtle borders |
| Small | 0px 1px 3px rgba(0,0,0,0.1) | Cards, slight elevation |
| Medium | 0px 4px 6px rgba(0,0,0,0.1) | Standard cards |
| Large | 0px 10px 15px rgba(0,0,0,0.1) | Modal overlays |
| XL | 0px 20px 25px rgba(0,0,0,0.1) | Important overlays |
| Elevation | 0px 8px 24px rgba(0,0,0,0.15) | Maximum depth |

### Shadow Usage

```
Card: shadow-md
Dropdown: shadow-lg
Modal: shadow-xl or shadow-elevation
Button Hover: shadow-sm
Input Focus: No shadow (border change)
```

## Interactive States

### Button States

**Default**
- Background: Aqua (#2FE7F2)
- Text: Matte Black (#0D1117)
- Border: None

**Hover**
- Background: Accent Dark (#1FB8C4)
- Text: Matte Black (#0D1117)
- Cursor: Pointer

**Active/Pressed**
- Background: Secondary Primary (#007BFF)
- Text: White (#FFFFFF)
- Box Shadow: Inner shadow

**Focus**
- Outline: 2px Aqua (#2FE7F2)
- Outline Offset: 2px
- Background: Unchanged

**Disabled**
- Opacity: 50%
- Cursor: Not-allowed
- Pointer Events: None

### Form Input States

**Default**
- Border: 1px Titanium Gray (#6C757D)
- Background: Matte Black (#0D1117)
- Text: White (#FFFFFF)

**Hover**
- Border: 1px Silver (#C5CBD3)
- Cursor: Text

**Focus**
- Border: 2px Aqua (#2FE7F2)
- Background: Unchanged
- Outline: None

**Filled**
- Border: 1px Electric Blue (#007BFF)
- Shows input has content

**Error**
- Border: 2px Error Red (#EF4444)
- Background: Error Red at 5% opacity

**Disabled**
- Opacity: 60%
- Background: Matte Black (#0D1117)
- Cursor: Not-allowed

## Animations & Transitions

### Duration Scale

| Speed | Duration | Usage |
| --- | --- | --- |
| Fast | 150ms | Hover, micro-interactions |
| Base | 300ms | Standard animations |
| Slow | 500ms | Page transitions, modals |

### Easing Function

**Standard Easing:** `cubic-bezier(0.4, 0, 0.2, 1)`
- Smooth, natural motion
- Used for all standard transitions

### Common Animations

```
Fade In: Opacity 0 → 100%, duration 300ms
Slide Up: Translate Y 20px → 0, duration 300ms
Scale In: Scale 0.95 → 1, duration 300ms
Fade Out: Opacity 100 → 0, duration 300ms
```

### Accessibility Note

Respect `prefers-reduced-motion` media query:
```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## Accessibility Standards

### Color Contrast Requirements (WCAG AA)

- **Large Text:** 3:1 ratio minimum
- **Normal Text:** 4.5:1 ratio minimum
- **Graphics:** 3:1 ratio minimum

### Contrast Examples

✅ **Pass**
- White (#FFFFFF) on Matte Black (#0D1117) = 21:1
- Light Gray (#E0E0E0) on Midnight Navy (#08131F) = 11.5:1

❌ **Fail**
- Silver (#C5CBD3) on Midnight Navy (#08131F) = 2.9:1
  - Solution: Use Light Gray instead

### Focus Indicators

- Always visible (never remove outline)
- Minimum 2px width
- Contrasting color (Aqua is standard)
- 2px offset from element

### Semantic HTML

- Use `<button>` for clickable elements
- Use `<a>` for links
- Use `<input>` with labels for forms
- Use heading hierarchy (`<h1>` → `<h6>`)
