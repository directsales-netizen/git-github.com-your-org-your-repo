# Premium TechNoir Design System

## Color Palette

### Background
- **Matte Black:** #0D1117
- **Midnight Navy:** #08131F

### Primary Accent
- **Aqua:** #2FE7F2

### Secondary Accent
- **Electric Blue:** #007BFF

### Neutral
- **Silver:** #C5CBD3
- **Titanium Gray:** #6C757D

### Text
- **White:** #FFFFFF
- **Light Gray:** #E0E0E0

## Typography

### Headings
- **Space Grotesk** (primary)
- **Sora** (secondary)

### Body
- **Inter** (all body text, UI, labels)

### Font Weights
- Headlines: 600-700 (bold, semi-bold)
- Subheadings: 600 (semi-bold)
- Body: 400-500 (regular, medium)
- Labels: 500 (medium)

## Logo Usage

### Icon
- Keep the shield badge almost exactly as designed
- Distinctive and memorable
- Flexible for all applications

### Wordmark
- Use cleaner, more premium typeface
- Dark navy or charcoal text on light backgrounds
- Aqua gradient version for dark backgrounds
- Optimized versions for: web, social media, invoices, business cards, packaging, apparel

## Design Principles

### Visual Language
- Modern
- Premium
- Minimal
- Clean
- Apple-inspired
- High-end
- Tech-focused
- Minimal clutter
- Strong visual hierarchy

### UI Components
- Rounded corners (8px-16px radius)
- Clean spacing (8px grid system)
- Subtle shadows and depth
- Smooth transitions (300-500ms)
- Accessible contrast ratios (WCAG AA minimum)

### Imagery
- Professional product photography
- High-quality, well-lit
- Consistent angle and perspective
- Clean backgrounds or contextual environments
- Show device condition grades clearly

## Brand Feeling

When customers interact with Premium TechNoir, they should feel:

- "I trust these people."
- "This looks professional."
- "They know what they're doing."
- "They're transparent."
- "They'll help me if I have a problem."

## Responsive Design

- Mobile-first approach
- Optimized for: 375px (mobile), 768px (tablet), 1024px (desktop), 1440px (wide)
- Touch-friendly interactive elements (minimum 44px tap targets)
- Performance-optimized for all devices

## Accessibility

- WCAG 2.1 AA compliance minimum
- Semantic HTML structure
- ARIA labels where needed
- Keyboard navigation support
- Color contrast ratios ≥ 4.5:1 for text
- Alt text for all images
- Captions for video content
